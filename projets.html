Création du site web de l’UEZ

Projet web institutionnel – Za-Kpota

Conception et mise en ligne du site officiel de l’Union des Étudiants de Za-Kpota (UEZ), conçu pour regrouper les informations institutionnelles, promouvoir les activités et fluidifier la communication entre les membres.

  • 🎯 Objectif : Centraliser les actualités, les documents et les projets de l’UEZ
  • 🧠 Compétences : UX/UI, HTML/CSS, structuration de contenu, communication
  • 🛠️ Outils : Visual Studio Code, Trello, Canva, GitHub
  • 💡 Résultat : Un site responsive, clair et institutionnellement valorisant
