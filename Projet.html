Mini-jeu interactif en JavaScript

Développement d’un jeu de devinette dynamique dans lequel l’utilisateur tente de découvrir un nombre généré aléatoirement. Le jeu affiche des messages interactifs selon les essais.

  • 🎯 Objectif : Mettre en pratique les bases du JavaScript (DOM, events, logique)
  • 🧪 Fonctionnalités : système de score, retour immédiat, relance automatique
  • 🛠️ Outils : HTML5, CSS3, JavaScript vanilla, Visual Studio Code
  • 📱 Résultat : Application responsive légère et intuitive, sans framework externe
